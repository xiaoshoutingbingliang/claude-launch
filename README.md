# claude-launch
极简API切换器：对于经常切换 API 密钥、管理不同项目和查找历史对话的小白用户来说，一个可视化的桌面启动能省去大量繁琐的命令行操作。 
---
功能齐全，双击运行：不仅能记住你的多个 API 账号（密钥和地址），还能记住你最近打开过的项目文件夹，并支持一键启动全新对话、继续上次对话或选择历史对话。
---

### 第一步：保存你的 API 账号（只需配置一次）

1. 在最上面的 **“1. API 账号配置与选择”** 区域：
   * **API 别名**：输入一个好记的名字，比如输入 `anyrouter`。
   * **接口地址 (Base URL)**：输入你使用的 API 地址 `https://anyrouter.top`。
   * **密钥 (API Key)**：输入以 `sk-` 开头的完整密钥。
2. 填写完这三项后，点击 **【💾 保存/更新当前账号】** 按钮。
3. **效果**：系统会弹窗提示保存成功。此后，你就可以随时在第一行的“快速切换账号”下拉菜单里一键选中它。如果你有其他网站的 API，也可以用同样的方法起个别名保存起来。

---

### 第二步：选择你要写代码的项目文件夹

1. 在中间的 **“2. 工作目录（项目文件夹）”** 区域：
   * 点击右侧的 **【📁 浏览...】** 按钮。
   * 在弹出的 Windows 文件夹选择器里，找到你放代码的项目文件夹，点击确定。
2. **效果**：当前项目路径会自动填入输入框，并且这个路径会被启动器记住。下次你再打开启动器时，可以直接在“历史项目目录”下拉菜单里快速选择，不需要重新寻找。

---

### 第三步：选择你的启动模式

根据你的需求，在 **“3. 启动模式”** 里勾选一项：

* **新建全新对话**：如果你要开始写一个全新的功能，或者新开一个文件，选这个。
* **直接继续上次对话**：如果你刚才在这个项目里写代码写到一半，临时关闭了终端，现在想直接**接着刚才的话题继续聊**，选这个。
* **列出历史对话并选择**：如果你想找回前几天或者更久以前和 Claude 在这个项目里的某次聊天记录，选这个。启动后终端会列出所有的历史聊天供你挑选。

---

### 第四步：一键启动

1. 确认上面三步都配置好后，点击最下方的 **【🚀 启动 Claude Code 编程助手】** 按钮。
2. 此时，你的桌面会弹出一个真正用来写代码的黑色命令行窗口，并自动开始启动 Claude Code。

> **💡 如果你是第一次运行 Claude Code：**
> 弹出的黑色窗口里会依次让你确认几项设置。你只需要看着屏幕，按以下步骤**连续按 Enter（回车键）** 即可：
> 1. 选择你喜欢的主题 $\rightarrow$ 按 **Enter** 确认
> 2. 确认安全须知 $\rightarrow$ 按 **Enter** 确认
> 3. 使用默认 Terminal 配置 $\rightarrow$ 按 **Enter** 确认
> 4. 信任工作目录 $\rightarrow$ 按 **Enter** 确认

看到绿色的输入提示符后，你就可以在终端里直接向 AI 提问、让它帮你写代码了！

---

### 🌟 以后使用有多方便？

下次当你打开电脑准备写代码时，你只需要：
1. 双击桌面上的 **`启动.bat`**。
2. 下拉菜单选择 **API 账号** $\rightarrow$ 下拉菜单选择 **项目路径** $\rightarrow$ 点击 **【启动】**。

全程只需要点击 3 下鼠标，不需要再复制任何密钥，也不需要输入任何 `cd` 或 `export` 命令行。
